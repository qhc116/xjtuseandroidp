# 服务器端
- 安装nodejs、redis-server、mongodb
```
//安装后启动服务
redis-server
mongod
```
- 在项目中需要npm安装nodejs组件express、multer、redis、mongoose、jwt-simple、crypto
```
npm install -S express
npm install -S multer
npm install -S redis
npm install -S jwt-simple
npm install -S express
npm install -S crypto
```
- 安装完成后，在项目所在文件夹启动
```
npm start
```
